# 📌# # carrotMarket
> 당근마켓

## ✔기능
<ul>
<li>Firebase Authenetication을 활용해 로그인,회원가입 구현 </li>
<li>회원가입 후 중고거래 아이템을 등록할 수 있다. </li>
<li>회원 기반으로 채팅 화면을 구현 할 수 있다. </li>
</ul>

##  ✔활용 기술
RecyclerView 
View Binding 
Fragment 
BottomNavigationView 
Firebase Storage 
Firebase Realtime Database 
Firebase Authentication
